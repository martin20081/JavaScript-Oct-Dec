# JavaScript-Oct-Dec

Welcome to the **JavaScript-Oct-Dec** repository!

## Overview

This repository is a comprehensive collection of JavaScript code samples, exercises, and learning materials created from October to December. The intent is to provide clear examples and structured practice for mastering fundamental JavaScript concepts.

## Repository Structure

- [`JS-syntax-fundamentals`](https://github.com/martin20081/JavaScript-Oct-Dec/tree/main/JS-syntax-fundamentals):  
  This directory contains resources focused on the fundamentals of JavaScript syntax. Here you will find tutorials, sample scripts, and practical exercises that cover essential programming concepts, coding patterns, and best practices in JavaScript.

_All files in this repository are written in JavaScript (100%), ensuring a focused and consistent learning experience._

## Key Topics Covered

- Variables and Data Types
- Operators and Expressions
- Control Flow (Conditionals & Loops)
- Functions and Scope
- Arrays and Objects
- Practice Problems and Solutions
